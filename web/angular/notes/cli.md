## ng generate

- Create component

```
$ng generate[g] component[c] <path/name>
```

- Create component without test files

```
$ng g c <path/name> --spec false
```

- Create a module

```
$ng g module[m] <path/name>
```

- Create a service

```
$ng g service[s] <path/name>
```
